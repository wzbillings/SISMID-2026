Welcome to “R Programming for Infectious Disease Modeling”!

This website contains the material for the 2026 Summer Institute in Statistics and Modeling in Infectious Diseases (SISMID) Module “R Programming for Infectious Disease Modeling”.

2026 course path

The core course covers Quarto, statistics in R, iteration, functions, functional programming, useful R packages, formulas and model objects, ODEs, and disease mapping/GIS. Exercise 1 begins on Day 1 and continues on Day 2; Exercise 2 begins on Day 2 and gives you structured time to build a short Quarto report.

Bootstrapping, Arrow, and power simulation are available as optional or take-home materials. General sample-size demonstrations and the advanced linear models draft are archived for 2026 rather than presented as active course modules.

Prerequisites

Participants are expected to have a basic understanding of R, including how to open and write R scripts, read in data, and use basic functions and syntax. Participants are also expected to have prior knowledge of basic descriptive statistics and regression modeling consistent with an introductory statistical course.

Before the course begins, you should install R and RStudio on your laptop. If you are using an older version of R, you should update it before the course begins. You will need at least R version 4.4.1 for this course, but using the most recent version is always preferable. You’ll also need Quarto.

  • You can install R from the CRAN website by clicking on the correct download link for your OS.
  • You can install RStudio from the Posit website.
  • You can install the latest version of Quarto from the Quarto website, but if you have a relatively up-to-date version of RStudio, it will also come with Quarto.

After installing, run the setup check. It confirms that R, Quarto, the main packages, and the QCRC data are ready before the first exercise.
