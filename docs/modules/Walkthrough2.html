Data science walkthrough 2: Repeating an analysis

Alex Edwards
Zane Billings

The same problem, repeated

In Walkthrough 1, we answered one country-level question:

In Pakistan since 2000, which years should we flag because MCV1 coverage was low and measles incidence was elevated?

Today we ask the next manuscript question:

Is Pakistan unusual, or do we see the same pattern across other countries in the region?

What changes today?

The analysis rule does not change:

Priority year = MCV1 coverage below 90% and at least 1 measles case per 100,000 people.

The workflow changes:

  1. Write the rule once.
  2. Test it on one country.
  3. Repeat it across many countries.
  4. Turn the repeated results into a table for the report.

Before we loop

Loops are easier when we know the exact task.

For one country, we need to:

  1. Keep rows for that country and year range.
  2. Calculate cases per 100,000 people.
  3. Apply the priority-year rule.
  4. Count the priority years.
  5. Return one row of results.

That is a function-shaped problem.

Function 1: apply the rule

add_priority_rule <- function(country_data,
                              coverage_cutoff = 90,
                              incidence_cutoff = 1) {
  country_data$cases_per_100k <- (
    country_data$measles_cases / country_data$population * 100000
  )

  country_data$priority_year <- ifelse(
    is.na(country_data$MCV1_coverage) | is.na(country_data$cases_per_100k),
    "missing information",
    ifelse(
      country_data$MCV1_coverage < coverage_cutoff &
        country_data$cases_per_100k >= incidence_cutoff,
      "priority year",
      "not priority"
    )
  )

  country_data
}

Function 2: summarize one country

summarize_priority_years <- function(country_name,
                                     data,
                                     min_year = 2000,
                                     coverage_cutoff = 90,
                                     incidence_cutoff = 1) {
  country_data <- data[
    data$country == country_name & data$year >= min_year,
  ]

  country_data <- add_priority_rule(
    country_data,
    coverage_cutoff = coverage_cutoff,
    incidence_cutoff = incidence_cutoff
  )

  priority_years <- country_data$year[
    country_data$priority_year == "priority year"
  ]

  most_recent_priority_year <- if (length(priority_years) == 0) {
    NA_integer_
  } else {
    max(priority_years)
  }

  max_cases_per_100k <- if (all(is.na(country_data$cases_per_100k))) {
    NA_real_
  } else {
    round(max(country_data$cases_per_100k, na.rm = TRUE), 2)
  }

  data.frame(
    country = country_name,
    region = country_data$region[1],
    sub_region = country_data$sub_region[1],
    years = nrow(country_data),
    priority_years = length(priority_years),
    missing_information_years = sum(country_data$priority_year == "missing information"),
    most_recent_priority_year = most_recent_priority_year,
    max_cases_per_100k = max_cases_per_100k
  )
}

Why this helps

Before writing a loop, we made one repeatable unit:

summarize_priority_years("Pakistan", data = measles)

If that works for Pakistan, the loop only needs to change the country name.

That is the main trick.

Repeat with a for loop

countries <- c("Afghanistan", "Bangladesh", "India", "Nepal", "Pakistan")

country_summaries <- data.frame()

for (country_name in countries) {
  one_summary <- summarize_priority_years(country_name, data = measles)
  country_summaries <- rbind(country_summaries, one_summary)
}

What did the loop do?

Each loop pass did the same job:

  1. Put one country name into country_name.
  2. Run summarize_priority_years() for that country.
  3. Add the one-row answer to the growing results table.

The computer is doing the repeated typing for us.

Which version should you use?

Use the version that helps you think clearly.

  • A for loop is often easier to read while you are learning.
  • lapply() is compact once the repeated task is already a function.
  • Both are better than copying and pasting the same analysis five times.

What goes in the report?

Methods sentence

We wrote a reusable function to classify each country-year as a priority year when MCV1 coverage was below 90% and measles incidence was at least 1 case per 100,000 people, then applied that function to every country in Asia.

Results sentence

The repeated analysis identified several countries in Asia with many priority years since 2000, suggesting that the Pakistan pattern was not unique under this simple rule.
