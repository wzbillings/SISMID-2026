Course materials and downloads

The most reliable option is to download the full course repository. That gives you the data, exercises, source files, slides, and project file in one folder.

  1. Go to the SISMID-2026 GitHub repository.
  2. Click Code.
  3. Click Download ZIP.
  4. Unzip the folder.
  5. Open SISMID-2026.Rproj.

Direct full-repository download: SISMID-2026 main branch ZIP

If you use Git:

git clone https://github.com/wzbillings/SISMID-2026.git

Individual data files

Use the full repository ZIP if you can. If you only need one data file, use these direct links:

Data file Download
QCRC COVID-19 ICU workbook QCRC_FINAL_Deidentified.xlsx
Measles vaccination and case data measles_final.Rds
Simulated city cases city-cases.csv
SIR observed data SIR-observed.csv
Mosquito surveillance data mos-nyc.csv

Save downloaded data files inside the repository’s data/ folder unless an exercise tells you otherwise.

For dataset descriptions and recommended uses, see the Dataset Catalog.

Exercises and report files

You can view exercises from the site navigation. To edit them locally, download the full repository ZIP or use these source-file links:

File Download source
Exercise 1 Exercise1.qmd
Exercise 2 Exercise2.qmd
QCRC starter template qcrc-mini-manuscript-starter.qmd
QCRC checklist and rubric qcrc-report-checklist-rubric.qmd

If a raw .qmd file opens in your browser, use Save As to save it into the repository’s exercises/ folder.

Slides and module files

The online slide decks are linked in the site navigation. There is not a separate PDF bundle in this repository right now.

For offline use:

  • download the full repository ZIP,
  • open SISMID-2026.Rproj,
  • use the .qmd files in modules/,
  • render locally if you need a fresh HTML copy.

If you want a PDF copy of a deck during class, use the browser print dialog or ask an instructor which output format is best for that deck.

Learner support pages

  • Setup check: confirm R, Quarto, packages, and data access.
  • Dataset catalog: choose and document course datasets.
  • Troubleshooting checklist: work through common R, Quarto, data, and model errors.
  • Model-reporting cheat sheet: report lm(), glm(), confidence intervals, odds ratios, and predictions clearly.
  • Loops/functions/apply guide: decide when to use vectorized code, a function, a loop, or an apply-family tool.

# Course materials and downloads

The most reliable option is to download the full course repository. That gives
you the data, exercises, source files, slides, and project file in one folder.

1. Go to the [SISMID-2026 GitHub repository](https://github.com/wzbillings/SISMID-2026).
1. Click **Code**.
1. Click **Download ZIP**.
1. Unzip the folder.
1. Open `SISMID-2026.Rproj`.

Direct full-repository download:
[SISMID-2026 main branch ZIP](https://github.com/wzbillings/SISMID-2026/archive/refs/heads/main.zip)

If you use Git:

```bash
git clone https://github.com/wzbillings/SISMID-2026.git
```
