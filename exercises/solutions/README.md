# Instructor Solution Structure

These files support the solution policy for Exercises 1 and 2.

- `Exercise1-solution.qmd`: Solutions for every required Exercise 1 task, plus
  outlines for advanced tasks.
- `Exercise2-solution.qmd`: Solutions for every required Exercise 2 task, plus
  outlines for advanced tasks.

The solution documents are instructor-facing exemplars, not a single required
student answer. Student choices may differ if the report renders, uses an
appropriate analysis, and interprets results cautiously.

Review before teaching:

- Confirm QCRC variable labels and endpoint definitions.
- Decide whether to distribute full solutions or only selected snippets.
- Update the examples if the workbook changes.
